# Installation

* Fork & Clone the repo
```
  git clone https://github.com/[username]/CirclEd.git
```

* Navigate through the project
```
  cd CirclEd
```
* Install all requirements
  ``` 
  pip install -r requirements.txt
  ```
  
* Run :
  ```
  python3 main.py
  ```
  
* Copy the localhost url (usually localhost:5000/) and paste in browser
